Shop App
- - - -

A web application that allows the customer to add selected products to cart. The cart functionanility is full CRUD.


Technologies Used
- - - -
* Frontend - HTML, CSS, Material CSS and Django(Opinionated Framework)
* Backend - Python
* Database - PSQL
* Deployment - Heroku
